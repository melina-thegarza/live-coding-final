Live Pulse Documentation

Welcome to the documentation for Live Pulse. This page provides information on how to use the Live Pulse live coding engine.

Getting Started

To get started with Live Pulse, follow these steps:

  1. Open the Live Pulse application.
  2. Enter your code in the provided textarea.
  3. Press the "re-evaluate" button to execute the code.

Code Syntax

The Live Pulse code syntax consists of commands for specifying rhythmic patterns of beats. Here are some examples:

  • 1@220: Plays a single beat at 220 Hz.
  • 2@330*2: Plays two beats at 330 Hz, each with a duration twice as long as the default.
  • 3*3@270: Plays three beats with a duration three times as long as the default, each at 270 Hz.

Advanced Features

Live Pulse also supports advanced features such as repeat operations. For example:

  • 3@340 2[1@220 2@330]: Plays three beats at 340 Hz followed by a sequence of two beats, each at different frequencies.
